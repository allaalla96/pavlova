A) How to add hover images? - top

Our theme support hover images for img tag. To add one, you must add .hover (class) for parent element of img tag and upload -hover.png image to the folder, where image is located. Caution! If image is using hover, it must be .png type. Both images must be .png type, otherwise it won't work.

B) How to change add slider images, and change text? - top

To add slider items, you must edit everything between line 136 to 139. If you want to add new slider item, you must copy one img tag and put it after last img tag. To insert text on slider, you must add img alt tag, you can insert text inside alt tag, it will show up on the slider. You are able to use html code inside too, but you must write it in with symbols. For example all symbols will start with & and end with ; , for example symbol > will be "& gt ;" (without spaces and quotes), opening bracket for html tags would be "& lt ;" (without spaces and quotes).
